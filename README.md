北洋年间，中国南部。一列马拉火车正在行驶，骗子县长马邦德正与汤师爷、夫人在车厢里宴饮高歌，正在此时，火车被一伙劫道的麻匪扔进了水里。众麻匪发现这车上无钱无货。匪首张麻子逼问马邦德钱在何处，马邦德急中生智，装成已淹死的师爷，自述是陪县长上任，到任后就有钱。张麻子遂伪装成县长马邦德，与“汤师爷”、夫人、一众兄弟一同上任鹅城。
